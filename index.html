O que é?

O Código Horus é uma linguagem de codificação simples, inspirada no Código Morse, que utiliza os símbolos binários 0 e 1 para representar letras, números e alguns sinais de pontuação. Foi desenvolvido como um projeto para explorar a criação de linguagens de codificação e tradutores em Java.

Regras Básicas

  • Utiliza apenas os símbolos 0 e 1.
  • Cada caractere (letra, número, pontuação) é representado por uma sequência única de 0s e 1s.
  • As sequências de caracteres individuais são separadas pelo símbolo / (barra).
  • As palavras são separadas pela sequência // (duas barras).

Estrutura do Projeto

O projeto segue o padrão arquitetural MVC (Model-View-Controller) e está organizado da seguinte forma:

  • src/HorusCodeTranslator.java (Model)

    Contém a lógica principal de codificação e decodificação do Código Horus.

  • src/HorusUI.java (View/Controller)

    Interface de linha de comando que gerencia a interação com o usuário.

  • src/HorusUtils.java (Utilitários)

    Funções utilitárias para operações de I/O e manipulação de área de transferência.

Como Usar o Tradutor Java

O tradutor é um programa de linha de comando escrito em Java que oferece várias funcionalidades. Para usá-lo:

  1. Compilação:

    Certifique-se de ter o JDK (Java Development Kit) instalado. Navegue até o diretório raiz do projeto (o que contém a pasta `src`) e execute o comando:

    javac src/*.java
  2. Execução:

    No mesmo diretório, execute o comando:

    java src.HorusUI

    O programa apresentará um menu interativo com várias opções:

Funcionalidades Disponíveis:

  • Codificar Texto para Código Horus: Converte texto comum em código Horus
  • Decodificar Código Horus para Texto: Converte código Horus de volta para texto
  • Carregar texto de arquivo: Lê texto de um arquivo e opcionalmente o codifica
  • Carregar código Horus de arquivo: Lê código Horus de um arquivo e opcionalmente o decodifica

Após cada operação, você pode:

  • Copiar o resultado para a área de transferência
  • Salvar o resultado em um arquivo
  • Voltar ao menu principal

Nota: Certifique-se de que seu terminal esteja configurado para usar UTF-8 para entrada e saída corretas de caracteres acentuados.

Exemplo de Uso

Texto Original:

Olá Mundo!

Codificação (Opção 1):

Input:

Olá Mundo!

Output (Código Horus):

111/0100/01//11/001/10/100/111/101011

Decodificação (Opção 2):

Input (Código Horus):

111/0100/01//11/001/10/100/111/101011

Output (Texto):

OLA MUNDO!

Opções Adicionais:

Após codificar ou decodificar, você pode:

  • Copiar o resultado para a área de transferência
  • Salvar o resultado em um arquivo de texto
  • Carregar conteúdo de arquivos para codificar/decodificar
